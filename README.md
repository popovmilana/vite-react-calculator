# vite-react-calculator

Kalkulator napravljen u React + Vite + TypeScript.

## Live verzija

https://vercel.com/popovmilanas-projects/vite-react-calculator

## Pokretanje lokalno
Pratite ove korake da biste pokrenuli projekat na svom računaru:

1.  Klonirajte repozitorijum:
    ```bash
    git clone https://github.com/popovmilana/vite-react-calculator.git
    ```
2.  Uđite u folder projekta:
    ```bash
    cd vite-react-calculator
    ```
3.  Instalirajte zavisnosti:
    ```bash
    npm install
    ```
4.  Pokrenite razvojni server:
    ```bash
    npm run dev
    ```
